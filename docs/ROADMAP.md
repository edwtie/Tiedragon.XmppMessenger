# Roadmap

## Alpha 1 - XMPP Core Chat

- XMPP account login
- TLS connection
- roster/contact list
- one-to-one chat
- presence: online, away, offline
- reconnect using stream management

Protocols:

- RFC 6120
- RFC 6121
- XEP-0030
- XEP-0198

## Alpha 2 - Conversation Quality

- message delivery receipts
- typing indicator
- message carbons for multi-device sync
- server-side message archive
- real-time text

Protocols:

- XEP-0184
- XEP-0085
- XEP-0280
- XEP-0313
- XEP-0301

## Beta 1 - Group And Files

- multi-user chat
- file upload
- image sharing
- notification polish
- first tester build

Protocols:

- XEP-0045
- XEP-0363

## Beta 2 - Security

- OMEMO end-to-end encryption
- encrypted media sharing
- local secure credential storage

Protocols:

- XEP-0384
- XEP-0454

## Beta 3 - Calls

- audio call signaling
- video call signaling
- WebRTC media transport
- STUN/TURN configuration
- real-time text during calls

- XEP-0166
- XEP-0167
